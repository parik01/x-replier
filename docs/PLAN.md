# План развития X-replier

## Этап 1 — надёжный MVP (готово в 0.2.0)

- Расширение Manifest V3 для X.
- Настройки OpenAI, промпта, лимитов и задержек.
- Анализ видимых постов, дедупликация и локальная история.
- Черновик и автоотправка как выбираемые режимы.
- Состояния очереди, проверка вставки, подтверждение результата, отменяемая пауза и глобальная блокировка.

## Этап 2 — управляемая очередь

- Панель кандидатов с текстом поста, черновиком и кнопками «принять / переписать / пропустить».
- Прокрутка ленты с ограничением числа проходов.
- Выбор источника: главная лента, поиск, список, профиль.
- Отчёт по обработке и экспорт журнала.

## Этап 3 — качество ответов

- Профили голоса и отдельные промпты для тем.
- Контекст треда и запрет нежелательных тем/слов.
- Проверка длины, языка и повторяемости черновика до вставки.
- Поддержка OpenAI Responses API и совместимых провайдеров.

## Этап 4 — надёжность

- Тесты чистых функций отбора постов.
- Адаптеры DOM-селекторов и диагностический режим.
- Миграции настроек и зашифрованное/системное хранение ключа, если платформа позволит.
- Мониторинг ошибок без передачи текста постов.
