# Архитектура X-replier 0.2.0

`background.js` — единственная часть, работающая с OpenAI API key. Он выполняет запросы к OpenAI, ограничивает повторные запросы, хранит глобальную блокировку задачи и обслуживает служебные команды popup/content script.

`content.js` работает в X: собирает только видимые карточки, создаёт очередь из простых данных (без долгоживущих DOM-ссылок), открывает редактор, проверяет вставку, показывает панель состояния и подтверждает результат отправки.

`core.js` содержит чистые функции: настройки, состояние, историю, очистку ответов, задержки и TTL блокировки. Эти функции покрыты unit-тестами.

История в `chrome.storage.local` разделяет `draftedPosts`, `sentPosts`, `skippedPosts`, `failedPosts` и `unknownPosts`. Только `sentPosts` увеличивает `sentToday`. API key также хранится локально и доступен только background worker; это удобно для личного использования, но не является защищённым секретным хранилищем.

Глобальная задача хранит `tabId`, `jobId`, `startedAt` и `heartbeatAt`. При зависании вкладки блокировка истекает через 30 секунд.
