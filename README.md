# X-replier

Chrome-расширение, которое находит подходящие видимые посты в X, генерирует для них ответы через OpenAI и вставляет текст в нативный редактор ответа.

## Что готово в MVP

- обработка только `x.com` и `twitter.com`;
- отбор видимых постов: без рекламы, репостов, ответов, собственных публикаций и дублей;
- дневной лимит, минимальная длина поста и случайная пауза;
- история обработанных post ID в `chrome.storage.local`;
- OpenAI Chat Completions с настраиваемыми моделью и системным промптом;
- два режима: **черновик** (по умолчанию) и **автоотправка**.

В режиме черновика расширение вставляет один ответ, а публикацию подтверждает пользователь. Автоотправка намеренно выключена по умолчанию: перед включением убедитесь, что промпт, лимиты и задержки соответствуют вашим правилам и правилам X.

## Установка

1. Скачайте или клонируйте репозиторий.
2. Откройте `chrome://extensions` и включите Developer mode.
3. Нажмите **Load unpacked** и укажите папку репозитория.
4. Откройте ленту `https://x.com/home`, нажмите иконку расширения, добавьте OpenAI API key и сохраните настройки.
5. Нажмите «Запустить».

## Ограничения MVP

X часто меняет DOM, поэтому селекторы могут потребовать обновления. Расширение работает с постами, уже отрисованными в текущей ленте; обход профилей, список кандидатов с предпросмотром и поддержка других LLM — следующие этапы из [плана](docs/PLAN.md).

## Конфиденциальность

Ключ и журнал хранятся локально в профиле Chrome. Текст выбранного поста отправляется в OpenAI только для создания ответа.
