# Changelog

## 0.2.0

- Добавлены отдельные состояния черновиков, подтверждённых отправок, пропусков, ошибок и неопределённых отправок.
- Дневной лимит увеличивается только после подтверждённой отправки.
- Добавлены проверка вставки текста, отменяемая задержка и блокировка параллельных вкладок.
- Добавлены проверка OpenAI, удаление API key, сброс истории и панель состояния на странице.
- Добавлены unit-тесты, архитектурная документация и сценарий ручного тестирования.
